# TrueCaddie App Scaffold

This directory is the future iOS app shell.

The first foundation slice adds a lightweight bundle inspector view scaffold.

It is still not a committed app target, but it now includes a `#Preview` that loads the published sample bundle from:

- `shared/sample-bundles/kungsbacka-nya.v1.json`

That gives us a source-controlled visual proof surface for the bundle contract before a full Xcode project is committed.
